## 📊 1042 vs 1042-S 快速对照
> **核心区别**：1042 是机构报给 IRS 的总表，1042-S 是发给你的个人凭证。

| 维度 | Form 1042 | Form 1042-S |
| :--- | :--- | :--- |
| **填报主体** | 高校、企业等付款机构 | 同左（出具给个人） |
| **接收对象** | 仅美国 IRS | 留学生个人 + IRS |
| **2026截止日** | 3月16日 | 3月16日 |

## 💡 为什么选择本项目？
* **SEO & 专业性**：针对 **1040-NR** 非居民税表深度优化。
* **逻辑透明**：基于 **Next.js** 开源构建，所有计算公式公开，确保隐私安全。
* **避坑指南**：避开漏报利息收入（如 1099-INT）导致的高额补税坑。
