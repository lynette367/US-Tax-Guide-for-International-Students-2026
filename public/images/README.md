# 页面插画图片说明

请将以下5张图片分别放入对应的文件夹，并命名为 `illustration.jpg`：

## 图片分配

1. **留学页面** (`/study`)
   - 文件夹：`public/images/study/`
   - 文件名：`illustration.jpg`
   - 图片描述：图书馆/学习场景（有窗户、书架、笔记本）

2. **国际志愿者页面** (`/volunteer`)
   - 文件夹：`public/images/volunteer/`
   - 文件名：`illustration.jpg`
   - 图片描述：市场摊位（水果和鲜花）

3. **互惠生页面** (`/aupair`)
   - 文件夹：`public/images/aupair/`
   - 文件名：`illustration.jpg`
   - 图片描述：天鹅湖场景（建筑、湖、天鹅）

4. **出国工作页面** (`/work`)
   - 文件夹：`public/images/work/`
   - 文件名：`illustration.jpg`
   - 图片描述：城市街道（水彩风格，有建筑和街灯）

5. **移民页面** (`/immigration`)
   - 文件夹：`public/images/immigration/`
   - 文件名：`illustration.jpg`
   - 图片描述：咖啡店外景

## 图片要求

- 格式：JPG 或 PNG
- 建议尺寸：宽度 1200-1600px，高度自适应
- 文件大小：建议控制在 500KB 以内以优化加载速度
